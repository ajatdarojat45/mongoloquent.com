| [`avg(column)`](/docs/tutorials/queries/read-data#avgcolumn)                                                           | Calculate the average value of a specific column.                            | `column: str`                                        |
| [`count()`](/docs/tutorials/queries/read-data#count)                                                                   | Count the number of documents matching the query criteria.                   | -                                                    |

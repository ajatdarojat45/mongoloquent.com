# Comparison operators

## Comparation operators table

| Operator | Mongo Operator | Description                         |
| -------- | -------------- | ----------------------------------- |
| `=`      | eq             | Equals                              |
| `!=`     | ne             | Not equals                          |
| `>`      | gt             | Greater than                        |
| `<`      | lt             | Less than                           |
| `>=`     | gte            | Greater than or equal to            |
| `<=`     | lte            | Less than or equal to               |
| `in`     | in             | In                                  |
| `notIn`  | nin            | Not in                              |
| `like`   | regex          | Like (case-insensitive) using regex |

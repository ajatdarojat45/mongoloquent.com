## Overview

Operators are used to filter data in queries. They allow you to specify conditions that the data must meet in order to be included in the results.

| Operator | Mongo Operator | Description                         |
| -------- | -------------- | ----------------------------------- |
| `=`      | eq             | Equals                              |
| `!=`     | ne             | Not equals                          |
| `>`      | gt             | Greater than                        |
| `<`      | lt             | Less than                           |
| `>=`     | gte            | Greater than or equal to            |
| `<=`     | lte            | Less than or equal to               |
| `in`     | in             | In                                  |
| `notIn`  | nin            | Not in                              |
| `like`   | regex          | Like (case-insensitive) using regex |
